# Codebase_Explainer_AI
An AI-based codebase analyzer that fetches GitHub repositories, summarizes the project, and generates structured explanations for key files.

It fetches repository files using the GitHub API and uses an AI model to:
- Summarize the overall project
- Explain key files
- Generate a structured report

---

## Features

-  Fetches repository data using GitHub API  
-  Generates project summary from README  
-  Explains important source code files  
-  Limits API usage to control cost  
-  Outputs clean, structured Markdown report  

---

## Tech Stack

- Python  
- GitHub REST API  
- OpenAI API  
- Requests Library
